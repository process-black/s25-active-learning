## LoopingVideo Features

The LoopingVideo component:
- Auto-plays and loops continuously
- Muted by default (browser requirement for autoplay)
- No controls by default (cleaner look)
- Auto-fits to markdown width (100%)
- Includes proper spacing and margins

## How It Works

Video, LoopingVideo, FloatingVideo, and InlineVideo components are automatically available in all MDX files thanks to the `/src/theme/MDXComponents.tsx` file that extends the global MDX scope.
